# CoRSIV 2.0 — Code Repository

This repository contains all analysis code associated with the manuscript describing CoRSIV 2.0 — an expanded catalogue of **Correlated Regions of Systemic Interindividual Variation** in human DNA methylation.

CoRSIVs are genomic loci at which DNA methylation level is (i) correlated across tissues within an individual and (ii) variable between individuals in a population. This repository provides the scripts and notebooks used to generate every figure in the paper, from initial CoRSIV screening through enrichment analyses and disease associations.

## Environment setup

### Python

This project uses [`uv`](https://github.com/astral-sh/uv) for environment management.

```bash
uv venv --python 3.11
source .venv/bin/activate
uv pip install -r requirements.txt
```

A `requirements.txt` listing pinned package versions is provided in this directory.

### R

R scripts were developed under **R ≥ 4.3**. Required packages are listed at the top of each script and can be installed with:

```r
install.packages(c("ggplot2", "dplyr", "circlize", "ComplexHeatmap", "VennDiagram", "remapR"))
```

### Command-line tools

Some steps require:

| Tool | Version tested | Purpose |
|---|---|---|
| `bedtools` | ≥ 2.30 | BED interval operations |
| `tabix` / `bgzip` | htslib ≥ 1.17 | Indexed BED access |
| `vcftools` | 0.1.16 | Tajima's D calculation |

## Data availability

Raw methylation data and large reference files are **not included** in this repository due to size and data-sharing restrictions. Input files should be placed in `data/raw/` before running any script. See the manuscript's Data Availability section for accession numbers and download instructions.

---

## Execution order

Figures build on each other in the following order:

1. **Figure 1** — run first; generates the CoRSIV BED files used by all downstream analyses.
2. **Figures 2–5** — largely independent once Figure 1 outputs are in place; see each figure's README for specific dependencies.
